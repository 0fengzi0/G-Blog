# 基于github的个人博客

修改配置文件```/static/Config.json```
```json5
{
  // 文章封面图api地址
  "pageCardImgServers": "https://api.9jojo.cn/acgpic",
  //你的用户名
  "username": "0fengzi0",
  // 抓取iss的仓库名
  "repo": "blog"
}
```
